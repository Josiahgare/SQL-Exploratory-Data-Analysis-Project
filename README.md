# SQL-Exploratory-Data-Analysis-Project
A comprehensive collection of SQL scripts for data exloration, analytics, and reporting. These scripts cover various analysis such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more.

It involves extracting facts and dimension tables from existing data warehouse then loading into a new database for exploratory data analytics (EDA) and advanced data analytics. Also, create two reports that contains VIEWs for easy accessing and querying by end-users for reporting and quick insights.
